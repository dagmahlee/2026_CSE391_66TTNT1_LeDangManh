- **Repository:** Nền tảng Web  

## 📘 Giới thiệu
Repository này được sử dụng để lưu trữ **toàn bộ bài tập, bài thực hành và project** trong quá trình học môn **Nền tảng Web**.  
Mục tiêu là nắm vững các kiến thức cơ bản về cách xây dựng và hoạt động của một website.

## 🛠️ Công nghệ sử dụng
- HTML
- CSS
- JavaScript
- Git & GitHub

## 📂 Nội dung repository
- Bài tập trên lớp  
- Bài thực hành  
- Project môn học (nếu có)

## 🎯 Mục tiêu
- Hiểu cấu trúc website
- Biết cách xây dựng giao diện web cơ bản
- Làm quen với quy trình quản lý mã nguồn bằng Git

## 📌 Ghi chú
Repository phục vụ cho mục đích **học tập**, không sử dụng cho mục đích thương mại.
